# Premium Food Ordering Website

A modern, premium food ordering website frontend built with React + Vite and Tailwind CSS.

## Features

- 🎨 Modern, premium UI/UX design
- ✨ Smooth animations and transitions
- 📱 Responsive design
- 🚀 Fast performance with Vite
- 🎯 Clean code structure

## Tech Stack

- React 18
- Vite
- Tailwind CSS
- React Router DOM

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

The app will be available at `http://localhost:5173`

### Build

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```
